# StockFlow Admin Dashboard

A professional admin dashboard for managing products, categories, and sales.

## Features

- Dashboard analytics
- Products management
- Categories management
- Sales tracking
- Search / Filter / Sort
- Responsive layout
- Charts and KPIs
- Loading / Error / Empty states

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router
- Context API
- Axios
- React Hook Form
- Zod
- Recharts
- Lucide React
